01The problem

This build is in draft. Content coming soon.

In the meantime, the pattern is straightforward: you give a Gemini Gem your full schema DDL, add a set of rules that explicitly tell it which tables and columns exist and forbid guessing, and the result is a SQL-writing assistant that hits ~90% accuracy on the first prompt. The details — the rules, the schema format, the edge cases — are what make the difference between 60% and 90%.
